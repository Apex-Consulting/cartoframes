### Embed map example

Use this iframe inside an `"example-map"` div container. Set a unique `id` as an attribute and assign your published map link in the `src` attribute.

```html
<div class="example-map">
    <iframe
        id="unique-ide"
        src="map-link-url"
        width="100%"
        height="500"
        style="margin: 20px auto !important"
        frameBorder="0">
    </iframe>
</div>
```
